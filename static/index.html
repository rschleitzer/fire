Features

FHIR R5 Compliant

Full support for FHIR R5 (5.0.0) specification with proper resource validation.

High Performance

Built with Rust and Axum for maximum performance and minimal latency.

Table-per-Resource

Optimized database design with separate current and history tables for each resource type.

ACID Transactions

PostgreSQL with full ACID compliance for reliable transaction bundles.

Search Parameters

Indexed search parameters for fast querying across resources.

Version History

Complete version history tracking with dedicated history tables.
